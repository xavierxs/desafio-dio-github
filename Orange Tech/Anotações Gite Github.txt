## **Algumas considerações e anotações após a conclusão do módulo de Introdução ao Git/GitHub de duração de 5 horas no Bootcamp Orange Tech na DIO.**

- O curso introdução ao Git/GitHub me possibilitou aprender desde instalar o Git na minha maquina até criar meu próprio repositório. Aprendemos acerca de comandos básicos para um  bom desempenho, objetos internos, configuração da chave SSH e token em prol da autenticação da máquina para o GitHub e como acontecem conflitos no GitHub e como resolvê-los.
- Passo a passo para criação de um repositório:
  - Criar um novo repositório pelo GitHub de preferência com README.md trazendo uma primeira impressão da ideia inicial do seu repositório com informações relevantes.
  - Reservar uma pasta em um local de sua preferência onde ficará salvo os arquivos do seu repositório após realização do próximo passo... 
  - Copiar o link HTTPS do repositório no GitHub
  - Usar o comando **git clone** para clonar os arquivos do GitHub na sua máquina através do Git.

Após todo esse passo a passo use sua criatividade para organizar seus estudos ou algo do seu interesse através de pastas e anotações.txt.
